# Tools

* Nmap – Network scanning tool used to discover hosts, open ports, services, and vulnerabilities.

* Hydra – Fast password brute-forcing tool for login services like SSH, FTP, HTTP, etc.

* Sqlmap – Automated tool to detect and exploit SQL injection vulnerabilities.

* WPScan – Security scanner specifically designed for WordPress websites.

* Nikto – Web server vulnerability scanner that checks for misconfigurations and outdated software.

* John (John the Ripper) – Password cracking tool used to crack hashes.

* Hashcat – Advanced and GPU-based high-speed password cracking tool.

* Metasploit – Exploitation framework used to develop and execute exploits against targets.

* Responder (LLMNR/NBT-NS) – Tool used to poison LLMNR/NBT-NS requests and capture NTLM hashes in local networks.

* Wireshark / Tcpdump – Network packet analysis tools used to capture and inspect traffic.

* Steghide – Tool used to hide or extract data inside image or audio files (steganography).

* OpenStego – Steganography tool to hide data inside images.

* QuickStego – Simple GUI tool for hiding and revealing secret messages in images.

* Dirb – Web content scanner used to brute-force hidden directories and files on websites.

* OleView – Windows tool to inspect OLE/COM objects and interfaces.

* Dcomcnf – Windows utility to configure Distributed COM (DCOM) settings.

* RpcDump – Tool used to enumerate RPC endpoints on Windows systems.

* Gacutil – Tool to manage assemblies in the .NET Global Assembly Cache (GAC).

* Searchsploit – Command-line tool to search exploits from the Exploit-DB database.

* sandfly-entropyscan – Linux malware detection tool that identifies suspicious high-entropy files.

* Strings – Tool that extracts readable text strings from binary files.

* FTK Imager – Digital forensics tool used to acquire and analyze disk images.

* Malwoverview – Malware analysis tool that extracts information from suspicious files.

* GDB / IDA – Debugging and reverse engineering tools used to analyze binary programs.

* Crunch – Tool used to generate custom wordlists for password attacks.

* Cewl – Tool that creates wordlists by crawling websites and extracting words.

* LinPeas – Linux privilege escalation auditing script.

* Aircrack-ng – Suite used to test and crack WiFi security (WEP/WPA).

* GTFOBINS – Online resource listing Linux binaries that can be abused for privilege escalation.

* VeraCrypt – Disk encryption software used to create encrypted volumes.

* Hashcalc – Tool used to calculate hash values (MD5, SHA1, etc.) of files.

* RainbowCrack – Password cracking tool that uses rainbow tables.

* Radare2 – Open-source reverse engineering framework for analyzing binaries.

* RockYou – Popular password wordlist commonly used for password cracking practice.

* SecLists – Collection of multiple wordlists used for fuzzing, brute force, and security testing.
